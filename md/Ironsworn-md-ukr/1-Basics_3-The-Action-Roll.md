# КИДОК ДІЇ

Коли ви робите хід, що представляє ризиковану або непевну дію, ви кидаєте три граники одночасно.

*   **Граники виклику:** Два десятигранники (d10)
*   **Граник дії:** Один шестигранник (d6)

```text
       / \             / \           +---+
      / 4 \           / 8 \          | 3 |
      \   /           \   /          +---+
       \ /             \ /
   ГРАНИКИ ВИКЛИКУ (d10)        ГРАНИК ДІЇ (d6)
```

Додайте вашу відповідну характеристику до вашого граника дії. Хід вкаже вам, яку характеристику додати, або може дати вам вибір. Деякі ходи скажуть вам використовувати одну з ваших шкал статусу, таку як здоров’я чи припаси, замість характеристики. Залежно від ходу або активів вашого персонажа, ви також можете мати можливість застосувати один або кілька бонусів, які називаються **додатками** (*adds*). Сума вашого граника дії, вашої характеристики та будь-яких додатків є вашим **значенням дії**. Ваше значення дії ніколи не може бути більшим за 10 — усе, що перевищує це число, ігнорується.

```text
  ГРАНИК  ХАРАКТЕ-  ДОДАТКИ    ЗНАЧЕННЯ
   ДІЇ    РИСТИКА                 ДІЇ
   +---+    +---+   +---+      _____
   | 3 | +  | 2 | + | 1 |   = (  6  )
   +---+    +---+   +---+      -----
```

Щоб визначити результат вашого ходу, порівняйте значення дії з кожним із граників виклику. Вам потрібно, щоб воно було більшим за індивідуальне значення цих граників.

```text
                   ГРАНИКИ ВИКЛИКУ
   ЗНАЧЕННЯ ДІЇ         / \             / \
      _____            / 4 \           / 8 \
     (  6  )  ------> / [v] \         / [x] \
      -----           \     /         \     /
                       \   /           \   /
                      Більше          НЕ більше
                        за                за
```

Є три можливі результати для ходу.

### [v][v] Точне влучання
Ваше значення дії більше за обидва граники виклику. Ви успішно робите те, що намагалися.

### [v][x] Ледь влучаєте
Ваше значення дії більше лише за один з граників виклику. Ви ймовірно досягли успіху, але з меншим ефектом або ціною.

### [x][x] Промах
Ваше значення дії не є більшим за жодний із граників виклику. Ви зазнали невдачі, або вам потрібно піти на серйозні поступки.

Хід вкаже вам, як трактувати результат вашої дії, або запропонує вибір. Цей результат може включати механічні зміни у статусі вашого персонажа та наративні зміни в поточній ситуації.

Коли ви зазнаєте промаху у ході, ви зазвичай побачите вказівку *Сплатити ціну*. Це спеціальний хід, який дозволяє вам обрати ймовірний негативний результат або кинути граники, щоб побачити, що сталося. Якщо ви граєте з ведучим (GM), він може проконсультуватися з цим ходом або просто сказати вам ціну.

Головне, що треба пам’ятати при промаху: завжди щось стається. Ситуація стає складнішою, драматичнішою або небезпечнішою. Щоб дізнатися більше про хід *Сплатити ціну*, дивіться [сторінка 105](3-Moves_7-Fate-Moves.md).

> [!IMPORTANT]
> **Нічиї завжди вирішуються на користь граників виклику.** Ваше значення дії має перевищувати — а не дорівнювати — граники виклику, щоб рахуватися за влучання.

## ДУБЛІ
Коли ви кидаєте граники для ходу, вам слід звертати увагу на **дубль** на граниках виклику. У кооперативній та соло грі це ваш тригер, щоб додати поворот, створити нове ускладнення або якось інакше змішати карти. Стається щось цікаве, несподіване або незвичне. Якщо ви не впевнені, *Спитайте Оракула*, що є ходом для встановлення відповідей на запитання або пошуку натхнення. Якщо ви граєте з ведучим (GM), дубль на граниках виклику може бути його сигналом запровадити несподіваний поворот подій.

```text
       / \             / \
      / 5 \     =     / 5 \
      \   /           \   /
       \ /             \ /
```

Результат дубля слід оцінювати залежно від результату вашого ходу.
*   **Точне влучання:** Дубль повинен означати поворот у розповіді, щось цікаве або нову можливість.
*   **Промах:** Дубль повинен означати посилений негативний результат, ускладнення або нову небезпеку. Все стає ще гірше для вас у несподіваний спосіб.

Ви також можете дозволити інтенсивності вашого успіху чи невдачі задавати тон того, як ви інтерпретуєте дубль. Викидання двох 10-к на ваших граниках виклику повинно спонукати вас ввести жахливий поворот подій або фатальну невдачу. Гірше просто не буває.

## ВЕДУЧИЙ ТА ГРАНИКИ
Якщо ви граєте в ролі ведучого (GM), ви можете зосередитись на скеруванні гри та відповідях на запитання й дії ваших гравців. Оскільки НПС (неігрові персонажі) не роблять ходів, вам не доведеться робити кидків дії. Проте, ви можете захотіти мати під рукою пару граників d10 для кидків Оракула [сторінка 22](1-Basics_9-Oracles.md#оракули).
